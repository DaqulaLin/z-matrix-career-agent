# Z-Matrix AI: The Universal Career Simulator

An autonomous career simulation agent powered by Google Cloud Platform (GCP). Z-Matrix projects 100,000 parallel trajectories by balancing Western management science with Eastern strategic wisdom, helping professionals navigate the AI transformation era across all professions.

## 🚀 Core Features
- **Dual-Engine Strategic Routing**: Dynamically balances Western Corporate Metrics (Game Theory, OKRs, KPIs) and Eastern Strategic Wisdom (Classical texts like *The Art of War*, *Guiguzi* written in 文言文).
- **Monte Carlo Simulations**: Runs 100,000 parallel universe projections to identify optimal career lifecycles and mitigation paths.
- **AI Replacement Risk Audit**: Delivers quantitative risk indexing and maps out hidden corporate promotion thresholds.
- **3D Evolution Roadmap**: Outlines structured actionable paths toward Niche Solopreneurship, Strategic Management, or High-Level Technical Leadership.

## 🛠 Tech Stack & Architecture
- **Vertex AI & Gemini 1.5 Pro**: Drives multi-step semantic reasoning, long-context grounding, and advanced dynamic prompt routing.
- **Google Cloud Run**: Hosts containerized Python microservices handling core agent reasoning loops and native canvas client rendering.
- **Cloud Run Jobs & Cloud Scheduler**: Orchestrating an automated data pipeline that continuously harvests global tech trends and job market nodes.
- **Client Application**: Native lightweight WeChat Mini Program ecosystem.

## 🔒 Security Notice
This public repository contains the open-source core business logic and agent architecture for the Google Cloud for Startups AI Agents Challenge. All production API credentials, database strings, and GCP Service Account keys are secured strictly via GCP Environment Variables and Secret Manager.
